# LIRI
Who is LIRI? LIRI is not who but a what: 
_Language_ Interpretation and Recognition Interface.

LIRI understands 4 commands: 

  * my tweets
  * spotify this song
  * movie this
  * do what this says
  * show log


LIRI was built with NodeJS and the following: 

  * npm packages:
    * cli-table
    * request
    * dotenv
    * inquirer
    
  * APIs:
  	* twitter
  	* spotify
  	* omdb (via request)
